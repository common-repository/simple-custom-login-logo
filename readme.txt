A simple plugin to customize the WordPress login page by adding a custom logo and changing the background color.

== Description ==

**Simple Custom Login Logo** is a lightweight and easy-to-use plugin that allows you to:
- Upload a custom logo for the WordPress login page.
- Change the background color of the login page using a color picker.

These changes make your WordPress login page more personalized and aligned with your brand.

= Features =
* Upload a custom logo using the WordPress media library.
* Change the background color of the login page.
* Easily manage settings via the WordPress admin area under Settings > Custom Login Logo.
* No coding required, just simple clicks to customize your login page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-login-logo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Settings > Custom Login Logo** in the WordPress admin panel.
4. Select or upload your custom login logo.
5. Use the color picker to set a custom background color for the login page.
6. Save your changes and visit your WordPress login page to see the customizations.

== Frequently Asked Questions ==

= Can I remove the custom logo or background color once I've set them? =

Yes! You can remove the custom logo by clicking the "Remove Logo" button in the plugin settings. You can also reset the background color to white using the color picker.

= Will this plugin slow down my website? =

No, the plugin is lightweight and only affects the login page. It does not add any overhead to your main website's front-end performance.

= Is the plugin compatible with multisite installations? =

Yes, the plugin can be used on multisite installations. However, each site will have its own settings.

= Can I add custom CSS to further customize the login page? =

The plugin focuses on the logo and background color, but you can add custom CSS to your theme's `functions.php` or via a custom plugin to further style the login page.
